# personal-portfolio
Personal Portfolio Website Built a responsive full-stack developer portfolio using HTML, Tailwind CSS, and JavaScript, featuring project showcases, smooth animations, and a contact form. Deployed on Netlify with zero build configuration.

# Full-Stack Developer Portfolio

A modern, responsive personal portfolio website built with:
- Tailwind CSS (CDN)
- Vanilla JavaScript
- HTML5

## 🚀 Deployment (Netlify)

1. Drag & drop the project folder into Netlify
OR
2. Connect your GitHub repo

### Settings:
- Build command: (leave empty)
- Publish directory: .

## ✨ Features
- Responsive design
- Smooth scrolling
- Animated sections (Intersection Observer)
- Contact form (frontend only)
- Dark theme UI

## 📌 Customize
- Replace "John Doe" with your name
- Update project links
- Add your real resume link
- Update social profiles
